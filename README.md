### Prerequisites

In order to run this project you need:

- `Ruby`
- `Rails`
- `Code editor (VSCode, RubyMine)`

### Setup

Clone this repository to your desired folder:

```sh
  cd my-folder
  git clone https://github.com/kerlos-bahaa/Blog-app
```

Install this project with:

```sh
  cd my-project
```
